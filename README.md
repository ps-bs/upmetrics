# UPMETRICS

## Backend

```
cd backend
```

### Installation
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev

```

### Backend GraphQL server will run
```
http://localhost:4000/graphql
```


## Frontend
```
cd frontend
```

### Installation
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Frontend app will run
```
http://localhost:8080/
```
